# pwnmachine

Vagrantfile for quickly (~50 mins) setting up a headless Ubuntu 18.04 VM for
exploit development on Linux.

```
sudo apt install vagrant
git clone https://github.com/kapaw/pwnmachine
cd pwnmachine
vagrant up
vagrant ssh
```
